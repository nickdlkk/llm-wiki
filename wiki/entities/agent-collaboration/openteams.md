# openteams

## 简介
openteams 是一个开源的多 Agent 协作框架，专注于让多个 AI 编程 Agent 以团队方式并行工作。多个 Agent 共享对话上下文，能够协作完成工程任务、内容生产等工作，显著提升复杂任务的处理效率。支持桌面应用与网络两种接入方式。

## 核心功能
- 多 Agent 并行协作：多个 AI Agent 同时工作，分解复杂任务
- 共享对话上下文：所有 Agent 在同一上下文中通信，保持任务连贯性
- 桌面应用支持：提供桌面客户端，开箱即用
- 网络接入：支持通过浏览器远程访问和协作
- 工程任务支持：适合代码开发、系统设计等工程类工作

## 优缺点
**优点：** 多 Agent 并行协作；共享上下文；桌面 + 网络
**缺点：** 新兴项目；生态待完善
