# T3code

**A minimal web GUI for coding agents** — currently supports Codex, Claude, and OpenCode.

## Features

- **Multi-Provider Support**: Codex, Claude, and OpenCode support
- **Web-based Interface**: Access coding agents from any browser
- **Easy Setup**: Run without installing via `npx t3`

## Supported Providers

- **Codex**: Install from OpenAI and run `codex login`
- **Claude**: Install Claude Code and run `claude auth login`
- **OpenCode**: Install from opencode.ai and run `opencode auth login`

## Quick Start

```bash
# Run without installing
npx t3

# Desktop app - install from GitHub Releases
```

## Installation

```bash
# Windows (winget)
winget install T3Tools.T3Code
```
