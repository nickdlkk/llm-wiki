# OpenCove

**An infinite canvas for Claude Code, Codex, terminals, tasks, and notes.**

## Features

- **Spatial Development Workspace**: Put AI agents, terminals, tasks, and notes on the same infinite 2D canvas
- **Full State Visibility**: Keep the full state of your work visible
- **Parallel Work**: See parallel work at a glance
- **Context Preservation**: Keep context visible, resume exactly where you left off
- **Multi-Agent Support**: Run multiple Claude Code or Codex sessions side by side

## Use Cases

- Running multiple `Claude Code` or `Codex` sessions side by side
- Keeping tasks and notes visible spatially
- Managing parallel AI agent workflows

## Tech Stack

Cross-platform desktop application (macOS, Windows, Linux)
