# TermCanvas

**Your terminals, on an infinite canvas** — An infinite canvas desktop app for visually managing terminals.

## Features

- **Infinite Canvas**: Arrange terminals on an unlimited 2D workspace
- **Visual Organization**: Position and organize terminals spatially
- **Focus & Zoom**: Navigate large terminal workspaces with ease
- **AI Agent Support**: Arrange multiple AI agents on the canvas
- **Hub View**: Knowledge graph view with sessions sidebar

## Screenshots

TermCanvas allows you to place multiple terminals and AI agents on an infinite canvas, keeping your work visible and organized spatially.

## Tech Stack

Cross-platform desktop application (macOS, Linux, Windows)
