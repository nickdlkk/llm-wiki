# NeZha

**An Agent-First IDE For Vibe Coding** — Claude Code + Codex, Git, editing, and task management, all in one place.

## Features

- **Multi-project Workspace**: Run multiple Claude Code and Codex agents across projects on your machine
- **Fast Switching**: Quickly switch between vibe coding tasks
- **Real-time Terminal**: Integrated terminal experience
- **Session Auto-discovery**: Automatically discover and manage agent sessions
- **Native Git Integration**: Complete Git workflow support
- **Lightweight**: Only 7MB installation package
- **Code Editor**: Built-in code editing capabilities
- **Real-time Usage Tracking**: Monitor agent usage in real-time

## Why NeZha?

Traditional IDEs like VS Code are designed for human developers. In the AI coding era, programming is becoming increasingly parallelized, but human attention remains limited. NeZha solves the challenge of tracking and managing tasks across multiple projects concurrently with an **Agent-First** philosophy.

## Tech Stack

- Desktop application with Claude Code and Codex integration

## Philosophy

NeZha is engineered with an **Agent-First** philosophy. It features a built-in terminal that directly integrates native Claude Code and Codex, so your AI coding agents can work in parallel instead of waiting in sequence.
