# Delight AI Agent

**Delight AI agent** is an enterprise-grade, omnichannel AI solution designed to revolutionize customer service by Sendbird. Unlike traditional chatbots, Delight AI agents understand intent, maintain context, take action, and adapt over time.

## Features

- **Omnichannel Support**: Integrates with SMS, email, WhatsApp, in-app chat, web, and social messaging platforms
- **Contextual Understanding**: Maintains conversation context across channels
- **Proactive Engagement**: Anticipates customer needs and initiates conversations
- **Unified Customer Insights**: Transforms omnichannel conversations into unified customer insights
- **Seamless Integrations**: Connects with Salesforce, Zendesk, Notion, Google Drive

## Repository Structure

- **JavaScript** (`js/`): React and CDN integration guides, interactive live examples, downloadable sample projects
- **iOS** (`ios/`): Swift integration guides and sample applications
- **Android** (`android/`): Android integration documentation

## Quick Start

1. Log in to your Delight AI dashboard
2. Go to **Channels → Messenger → Basic information** to check Application ID and AI Agent ID
3. Integrate the appropriate SDK into your application
4. Initialize with Application ID and Agent ID

## Platform

Enterprise customer service AI by Sendbird
