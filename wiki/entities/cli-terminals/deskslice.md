# DeskSlice

## 简介
DeskSlice is a local network solution that enables low-latency video streaming and input injection via WebRTC. It is designed for remotely controlling real Codex or similar panels within VS Code, specifically optimized for mobile phone usage over LAN connections.

## 核心功能
- WebRTC low-latency video streaming
- Input injection for remote control
- VS Code Codex panel control
- Mobile-optimized LAN usage
- Private and secure local network communication

## 优缺点
**优点：** WebRTC 低延迟；手机远程控制；局域网私密安全
**缺点：** 局域网专用；需要额外设置
