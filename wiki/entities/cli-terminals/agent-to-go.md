# Agent Governance Toolkit (agent-to-go)

**AI Agent Governance Toolkit** by Microsoft — Policy enforcement, zero-trust identity, execution sandboxing, and reliability engineering for autonomous AI agents.

## Features

- **Policy Enforcement**: Define and enforce policies for AI agent behavior
- **Zero-Trust Identity**: Identity management for agents
- **Execution Sandboxing**: Safe execution environment for agent actions
- **Reliability Engineering**: Built-in reliability patterns for autonomous agents
- **OWASP Agentic Top 10**: Covers 10/10 of the OWASP Agentic Top 10 security concerns

## Compliance

- **OWASP Agentic Top 10**: Full coverage (10/10)
- **OpenSSF Scorecard**: Passes OpenSSF best practices

## Tech Stack

- Python
- Node.js (npm SDK)
- .NET (NuGet)
- MIT License

## Installation

```bash
# PyPI
pip install agent-governance-toolkit

# npm
npm install @microsoft/agent-governance-sdk

# NuGet
dotnet add package Microsoft.AgentGovernance
```
