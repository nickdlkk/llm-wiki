# phoneshell

## 简介
phoneshell is a cross-platform terminal tool that supports remote access. It provides remote terminal functionality across multiple platforms, allowing users to access and control remote systems from their devices.

## 核心功能
- Multi-platform support
- Remote terminal access
- Cross-device connectivity
- Basic remote shell functionality

## 优缺点

**缺点：** 功能相对基础
