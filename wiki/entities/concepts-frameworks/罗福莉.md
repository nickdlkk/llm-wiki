# 罗福莉（Luo Fuli）

## 背景

- 前阿里达摩院 NLP 研究者
- 前 DeepSeek 核心贡献者（参与过几代核心模型）
- 现**小米大模型团队负责人**，主导 MiMo-V2 系列
- 媒体称"AI天才少女"，本人不喜欢这个称呼

## 核心观点

### 关于 OpenClaw
春节三天高强度使用后认知发生巨变：从"有灵魂的产品"到"研究助手"到"数字分身"。认为 OpenClaw 是划时代的 Agent 框架，核心是框架设计弥补模型短板。

### 关于 AGI
> "我认为 Mimo-v2-Flash 达到了 AGI 的 20%，2-3年内会实现 AGI。"

### 关于团队
- 小米 100人含所有链路（数据/Infra/Post-train/开发/产品/算法）
- 实习生比例很高
- 每天 11点-凌晨 12-14点，睡眠 4-6小时（个人生理特点）

## 代表言论

- "环境反而比经验更重要"
- "一个非常复杂的 Agent 框架设计，能够弥补非常多模型能力的短板"
- "想象力是沉积的"
