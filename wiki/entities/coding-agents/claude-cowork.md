# Claude Cowork (Agent Cowork)

## 简介
桌面 AI 助手 GUI 工具，强力兼容 Claude Code 完全相同的配置（~/.claude/settings.json），可使用任意兼容 Anthropic API 的模型，为用户提供图形化的 AI 编码辅助体验。

## 核心功能
- Claude Code 配置兼容：完全兼容 ~/.claude/settings.json 配置
- 任意模型支持：可使用任意兼容 Anthropic API 的模型
- 桌面 GUI 界面：提供图形化的操作界面
- 配置复用：无需重新配置即可迁移现有 Claude Code 设置

## 优缺点

**优点：**
- Claude Code 配置兼容：无缝迁移现有配置
- 任意模型支持：灵活选择 AI 模型

**缺点：**
- 桌面 GUI 相对重型
