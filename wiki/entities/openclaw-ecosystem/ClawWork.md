# ClawWork

## 简介
ClawWork 是一个开源的桌面应用和 PWA（渐进式网页应用）客户端，专为 OpenClaw 生态设计。它用于连接用户自建的 OpenClaw 网关，将散落的多任务、多会话智能体工作转化为结构化的「工作区」，避免传统聊天记录的碎片化问题。

## 核心功能
- OpenClaw 深度集成：专为 OpenClaw 网关打造的客户端
- 工作区结构化：将多任务、多会话组织为清晰的工作区
- 桌面应用支持：提供原生桌面客户端，体验流畅
- PWA 支持：支持渐进式网页应用，可在浏览器中随时访问
- 多会话管理：高效管理多个智能体会话和任务

## 优缺点
**优点：** OpenClaw 深度集成；工作区结构化；桌面 + PWA
**缺点：** 强依赖 OpenClaw
