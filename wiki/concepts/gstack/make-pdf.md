# make-pdf

Turn any markdown file into a publication-quality PDF. Proper 1in margins, intelligent page breaks, page numbers, cover pages, running headers, curly quotes and em dashes, clickable TOC, diagonal DRAFT watermark.

## When to Use

Use when asked to "make a PDF", "export to PDF", "turn this markdown into a PDF", or "generate a document".

## Voice Triggers

- make this a pdf
- make it a pdf
- export to pdf
- turn this into a pdf
- turn this markdown into a pdf
- generate a pdf
- make a pdf from
- pdf this markdown
