# landing-report

Read-only queue dashboard for workspace-aware ship. Shows which VERSION slots are currently claimed by open PRs, which sibling Conductor workspaces have WIP work likely to ship soon, and what slot /ship would pick next.

## When to Use

Use when asked to "landing report", "what's in the queue", "show me open PRs", or "which version do I claim next".

## Triggers

- landing report
- version queue
- ship queue
- what version comes next
- show open PR versions
