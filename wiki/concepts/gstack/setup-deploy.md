# setup-deploy

Configure deployment settings for [[land-and-deploy]]. Detects your deploy platform (Fly.io, Render, Vercel, Netlify, Heroku, GitHub Actions, custom), production URL, health check endpoints, and deploy status commands.

Writes the configuration to CLAUDE.md so all future deploys are automatic.

## When to Use

Use when: "setup deploy", "configure deployment", "set up land-and-deploy", "how do I deploy with gstack", "add deploy config".
