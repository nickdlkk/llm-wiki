# plan-design-review

Designer's eye plan review — interactive, like CEO and Eng review. Rates each design dimension 0-10, explains what would make it a 10, then fixes the plan to get there.

## When to Use

Works in plan mode. For live site visual audits, use /design-review instead.

Use when asked to "review the design plan" or "design critique".

Proactively suggest when the user has a plan with UI/UX components that should be reviewed before implementation.
