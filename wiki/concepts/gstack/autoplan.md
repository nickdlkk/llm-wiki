# autoplan

Automated multi-phase plan review pipeline that chains together CEO, Design, Engineering, and Developer Experience reviews in sequence.

## Phases

1. **Phase 1: CEO Review** — Strategic alignment, 10-star product thinking
2. **Phase 2: Design Review** — UI/UX scope detection (only runs if UI scope detected)
3. **Phase 3: Engineering Review** — Architecture, data flow, edge cases
4. **Phase 3.5: DX Review** — Developer experience (optional, skipped if no DX scope)

## When to Use

Use when you want a comprehensive plan review covering all four dimensions without running each review manually.
