# 生产力变革时代

> "26年的主要旋律是生产力变革，高生产力场景的持续突破。"
> — 罗福莉，2026-04-24

## 核心定义

2026 年是从"预训练主导的 Chat 时代"转向"后训练、Agent 框架、系统工程主导的 Agent 时代"的拐点。

## 两个演变方向

### 高价值路径
追求更强的智能，替代顶尖人群的任务：
- 更长的 Context
- 更多的 Token 消耗量
- 能替代最顶尖那一群人的智能就够了
- "下面那群人的智能可以慢慢以另外的方式替代掉（比如 Robotics）"

### 社会普适路径
做一个对全社会更有益、对所有人都有帮助的模型：
- 更注重多模态（尤其是对视频的理解、对细微环境的理解）
- 注重成本——成本是产生革命非常重要的因素
- 必须达到足够高的替换系数比

## 成本革命的前提

> "花十块钱的 API 就能干完 1000 块钱的事情，你肯定很愿意用。如果你的 API 贵十倍或几十倍，差距很小，你就会很排斥用这样一套复杂的东西去优化。"

## 多 Agent 协作

- 更长程的任务需要 Multi-Agent 之间的协作（单一 Agent 做不了复杂任务）
- Multi-Agent 能提升效率（任务完成速度）和节省成本
- 目前尚未看到 Multi-Agent 能实现更高上限的证据，但会持续发展

## 人的意义

> "Agent 能做什么样的高价值任务变得更重要了。这时候人更应该思考自己的意义和价值是什么。"

当 Agent 能替代大部分工作时，人需要转向更高阶的价值创造——架构设计、澄清需求、参与业务逻辑。
