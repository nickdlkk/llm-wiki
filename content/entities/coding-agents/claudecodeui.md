# ClaudeCodeUI

## 简介
ClaudeCodeUI 是 Claude Code 的桌面图形用户界面客户端，为开发者提供可视化的项目管理和操作环境。它在保留 Claude Code 强大编码能力的同时，通过桌面应用的形式让项目级管理更加直观便捷，适合不习惯命令行的团队成员使用。

## 核心功能
- **桌面客户端**：独立的桌面应用程序，无需终端即可操作 Claude Code
- **项目级管理**：支持多项目切换、项目目录可视化管理、文件浏览器集成
- **可视化任务面板**：图形化展示 Agent 任务队列、进度和历史记录
- **会话管理**：支持保存、导入、导出对话记录，便于追踪项目历程
- **集成开发辅助**：内置文件编辑预览、终端输出可视化等功能

## 优缺点
**优点：** 桌面客户端开箱即用；项目级管理更直观；适合团队协作场景

**缺点：** 需要独立安装；相对命令行版本可能存在功能差异；资源占用较高
