# agentc

## 简介
为 AI 编码代理提供容器级隔离与持久化工作环境的工具。通过隔离容器运行 Claude Code、GitHub Copilot CLI 等工具，适合多项目并行开发场景，避免环境冲突。

## 核心功能
- 容器级隔离：为每个项目提供独立的容器环境
- 多项目并行：支持多个项目同时开发，环境互不干扰
- 持久化工作环境：工作状态和上下文可持久保存
- 支持多种 AI 编码工具：Claude Code、GitHub Copilot CLI 等
- 适合复杂开发环境：解决多项目并行时的依赖冲突问题

## 优缺点

**优点：**
- 容器级隔离：环境完全独立，避免依赖冲突
- 多项目并行：支持同时开发多个项目
- 持久化工作环境：保持工作状态连续性

**缺点：**
- 需要 Docker：依赖 Docker 环境
- 配置有一定复杂度：需要一定的学习成本
