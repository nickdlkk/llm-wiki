# agentboard

## 简介
agentboard provides a web-based graphical user interface for managing tmux sessions. It enables users to monitor and control tmux sessions through a browser, making it particularly useful for AI agent session management and remote administration.

## 核心功能
- Web UI for tmux session management
- Remote access via browser
- Visual session monitoring and control
- Support for AI agent workflow integration

## 优缺点
**优点：** Web UI for tmux；远程访问；可视化会话管理
**缺点：** 依赖 tmux；Web 界面相对简单
