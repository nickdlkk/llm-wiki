# Mini-Term

**为 AI 时代打造的桌面终端管理器** — A desktop terminal manager for the AI era, built with Tauri v2.

## Features

- **多项目 (Multi-Project)**: Organize terminals by project
- **多标签 (Multi-Tab)**: Tab-based terminal management
- **分屏布局 (Split Layout)**: Split-screen terminal layouts
- **AI 进程感知 (AI Process Awareness)**: Know when AI agents complete

## Problem Solved

1. **重量级工具多余**: All In AI users only need terminal for running Agents, but VS Code / IDEA are heavy and memory-intensive
2. **多 Agent 并发无感知**: Running multiple Claude/Codex sessions, no intuitive way to see when an Agent finishes
3. **项目切换不便**: System terminals lack multi-project organization, tabs, and split management

Mini-Term solves all these problems with a lightweight desktop application.

## Tech Stack

- Tauri v2
- React 19
- Rust 2021
