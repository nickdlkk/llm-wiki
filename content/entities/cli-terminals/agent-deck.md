# agent-deck

## 简介
agent-deck 是一款专为 AI 编码代理设计的终端会话管理 TUI（文本用户界面）「指挥中心」。它提供统一的会话列表、状态可视化和管理功能，支持 Claude Code、Gemini CLI、OpenCode、Codex 等多种 AI 编码代理，帮助开发者高效管理多个 AI 代理会话。

## 核心功能
- 多 Agent 统一管理，支持多种 AI 编码代理
- TUI 高效界面，快速切换和管理会话
- 会话状态可视化，实时监控代理状态
- 统一的指挥中心体验
- 支持 Claude Code、Gemini CLI、OpenCode、Codex 等主流 AI 代理

## 优缺点
**优点：** 多 Agent 统一管理；TUI 高效；会话状态可视化

**缺点：** TUI 学习曲线
