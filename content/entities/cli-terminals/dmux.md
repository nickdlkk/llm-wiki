# dmux

## 简介
dmux 是一款将 tmux 终端复用功能与 git worktree 深度集成的开发工具，专为需要并行处理的开发者设计。它允许用户在多个终端会话中同时管理多个 git worktree，大幅提升并行开发效率，特别适合需要同时处理多个功能分支或修复的开发者。

## 核心功能
- tmux + worktree 深度集成，统一管理终端和 git 工作树
- 支持并行开发，同时处理多个分支
- 高效的会话管理和切换
- 简化多分支开发工作流
- 增强的 git 工作流可视化

## 优缺点
**优点：** tmux + worktree 集成；并行开发支持

**缺点：** 相对小众
