# IM-Hub

**Universal messenger-to-agent bridge** — connect WeChat/Feishu/Telegram to Claude Code/Codex/Copilot/OpenCode, or any custom agent via ACP.

## Features

- **Multi-Messenger Support**: WeChat, Feishu (Lark), Telegram
- **Multi-Agent Support**: Claude Code, Codex, Copilot, OpenCode, or any ACP-compatible agent
- **Bidirectional Communication**: Send tasks from messenger apps to AI agents and receive responses
- **Session Management**: Manage multiple agent sessions
