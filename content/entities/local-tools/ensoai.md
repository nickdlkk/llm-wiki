# EnsoAI

**Multiple Agents, Parallel Flow** — Unleash parallel intelligence within a single project. Let Claude, Gemini, and Codex weave through different worktrees simultaneously without context switching.

## Features

- **Parallel AI Agents**: Multiple AI coding agents working simultaneously across different git worktrees
- **Agent-First IDE**: Built for the era of vibe coding where AI generates code
- **Git Worktree Integration**: Each branch treated as first-class workspace with dedicated AI context
- **Multi-Provider**: Supports Claude, Gemini, and Codex

## Workflow

Stop stashing and popping. EnsoAI treats every branch as a first-class workspace with its own dedicated AI context.

## Tech Stack

- Electron 39+
- React 19
- TypeScript 5.9

## Philosophy

Traditional IDEs like VS Code are designed with the human developer at center. As AI takes over more code generation, programming becomes increasingly parallelized. EnsoAI solves the challenge of rapidly tracking and managing tasks across multiple projects concurrently with an **Agent-First** philosophy.
