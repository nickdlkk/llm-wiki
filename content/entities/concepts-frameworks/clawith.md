# Clawith

**Clawith** — "OpenClaw for Teams" — 是开源多 Agent 协作平台，让多个 AI Agent 和人类像团队成员一样一起工作。

GitHub: [dataelement/Clawith](https://github.com/dataelement/Clawith) — 3.5k ⭐, Python

## 定位

将 OpenClaw 从单 Agent 工具扩展为**团队协作平台**：多个 Agent 拥有持久身份、共享记忆和社交网络。

## 核心特点

- **OpenClaw 生态**：基于 OpenClaw 构建，复用其技能和工具生态
- **多 Agent 身份**：每个 Agent 有持久身份，不是临时工具
- **共享记忆层**：Agent 之间共享上下文和记忆
- **团队协作**：支持频道、任务分配、进度追踪

## 技术栈

- **语言**: Python
- **生态**: OpenClaw plugin 系统
