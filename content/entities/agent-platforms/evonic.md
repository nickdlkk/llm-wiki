# Evonic

## 简介
Open Agentic AI Platform，支持多 agent 编排、Workplace 远程执行、Heuristic 安全检测。Agent 六维度：Concept/Model/Tools/Knowledge Base/Channel，为企业级 AI Agent 部署提供完整的基础设施支持。

## 核心功能
- 多 agent 编排：支持多个 AI Agent 协同工作，实现复杂任务的分解与执行
- Workplace 远程执行：支持 Local/Remote SSH/Cloud Evonet 多种部署模式
- Evonet 云端执行：无需公网 IP/SSH/防火墙即可实现远程 agent 执行
- Heuristic 安全检测：智能安全分析，保障 agent 操作安全性
- 多租户支持：适合企业环境下的多团队协作
- Agent 六维度抽象：Concept/Model/Tools/Knowledge Base/Channel 完整维度

## 优缺点

**优点：**
- Workplace 抽象（Local/Remote SSH/Cloud Evonet）提供灵活的部署选项
- Evonet 无需公网 IP/SSH/防火墙，降低部署复杂度
- 多租户支持，适合企业级应用

**缺点：**
- v0.1.8 项目较早期，稳定性有待验证
- Discord/Slack 集成仍是规划状态
