# openagents

## 简介
OpenAgents 是一个开放的 AI 代理协作平台，旨在让 AI 代理与人类用户在共享工作空间中无缝协作。用户可以安装任意编码代理，连接到工作区，然后通过浏览器进行协同工作。这种浏览器内协作模式降低了使用门槛，无需复杂配置即可开始使用。

## 核心功能
- 浏览器内协作：在浏览器中直接与 AI 代理协同工作，无需安装额外软件
- 多 Agent 支持：支持同时运行多个不同类型的 AI 代理
- 工作区共享：团队成员可以在同一工作区中查看和参与协作
- 编码代理集成：可安装并使用各种编码代理进行开发任务
- 自托管选项：支持私有部署，保障数据隐私

## 优缺点
**优点：** 浏览器内协作；多 Agent 支持；工作区共享
**缺点：** 浏览器环境限制；自托管需要配置
