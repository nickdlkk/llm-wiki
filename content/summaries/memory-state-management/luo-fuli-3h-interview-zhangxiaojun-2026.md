# 罗福莉3.5h访谈摘要：AI范式已然巨变

## 核心判断（四个）

1. **算力分配 3:1:1**（研究:预训练:后训练）。研究是主菜不是配菜；预训练和后训练算力相当是行业拐点信号。
2. **Agent 入场圈 = Opus 级**。达不到 Opus 水平做 Agent 是玩具。
3. **长上下文稳定性只有 Opus + Sonnet 真能用**。别看宣传看实战。
4. **长任务训练物理上不可行**。业界用短训 + 激活迂回，不是直接训长任务。

## OpenClaw 三天认知转变

- **Day 1**：凌晨两点到六点，感受到"有灵魂"——会提醒睡觉（System MD 拼时间到 Context）
- **Day 2**：变成数字分身——团队管理、筛选人才都用它
- **Day 3**：两小时做出 User Agent——用于生成 Agent 场景数据做 SFT/RL

## 关键洞见

- Agent 框架是人和模型之间的"厚中间层"，比前端 UI 重要得多
- 复杂 Agent 框架设计能弥补模型能力的短板
- OpenClaw 开源可改，Claude Code 是黑盒
- AGI 20%，2-3年内实现（罗福莉个人判断）
